The program XMLOperations has been built and tested in Visual Studio 2015 in Python version 2.7

It was also tested on the command prompt using Python 3.6.

In order to run from the command line, please run the below command from the folder where you
pulled the project from Github

python.exe .\XMLOperations.py <path to input folder> <path to output ffolder>

You can also load the project in VS by opening the project XMLOPerations.pyproj from the folder where you
pulled the project from Github. You can run the unit tests there and also execute the program from within VS.
